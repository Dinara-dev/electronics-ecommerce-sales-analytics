# 🛍️ Electronics E-Commerce Sales Analytics Dashboard

## 📊 Overview
Power BI dashboard analyzing electronics retail sales,
brand performance, product category distribution,
and weekly/monthly revenue trends.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Excel / CSV data source

## 📌 Key Metrics
- Total Revenue: 349 MLN so'm
- Total Sales: 60K
- Average Check: 5.80 K so'm
- Top Category: Веб-камера (87 MLN)
- Top Brand: A4Tech

## 📈 Dashboard Features
- Monthly revenue trend (line chart)
- Category revenue breakdown (donut chart)
- Brand price comparison (bar chart)
- Weekly sales matrix by month
- Customer level segmentation (Basic / Premium / Mid)
